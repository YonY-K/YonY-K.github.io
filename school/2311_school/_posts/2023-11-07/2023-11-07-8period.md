### 문제
1~20사이의 무작위 정수 10개를 추출하여 배열에 저장한다.<br>
중복되지 않도록 해야 한다.<br>
숫자를 하나 뽑고 그 숫자가 배열에 이미 저장되어 있지 않은지 검사한다.<br>

```java
//랜덤 객체 생성
//배열 선언
//boolean 변수 선언
//for문1 : 랜덤 숫자 뽑기
//for문2 : 랜덤 숫자가 배열에 들어있는지 검사 boolean 리턴?
//for문2에서 중복이 있다면 break하고 다시 돌리기
//중복이 없다면 배열에 저장
//배열 속 숫자를 저장하기 위해 String 변수 하나 선언하고 배열 속 숫자 넣어주기
//출력

Random rd = new Random();
int[] arr = new int[10];

for(int i=0,n=0;i<10; ) {
    n = rd.nextInt(20)+1;
    boolean ok = true;
    
    for(int j=0;j<=i;j++) {
        if(n==arr[j]) {
            ok = false;
            break;
        }
    }
    
    if(ok) {
        arr[i++]=n;
    }
}

String s = "";

for(int i=0;i<arr.length;i++) {
    s += arr[i]+" ";
}

System.out.println(s);
```

//선생님 해답
Random rd = new Random();
int[] arr = new int[10];
int filled = 0;
int tmp = 0;

while(filled<10) {
    tmp = rd.nextInt(20)+1;
    boolean duplicated = false;
    for(int i=0;i<=filled;i++) {
        if(arr[i]==tmp) {
            duplicated = true;
            break;
        }
    }
    if(!duplicated) {
        arr[filled++]=tmp;
    }
}
