- countinue문 사용
  - continue가 실행되면 그 아래 문장은 실행되지 않고 루프를 다시 반복하게 된다.

```java
Random rd = new Random();
for(int cnt=0, num=0; cnt<10; ) {
    num = rd.nextInt(20);
    if(num%2!=0) continue;
    
    //continue가 실행되면 이 문장은 실행 안됨
    System.out.printf("%d. %d\n", ++cnt,num);
    
}
```
