## Test.vue

```html
<template>
  <h1>{{msg}}</h1>
</template>

<script>
export default {
    name: 'test',
    data() {
        return {
            msg :'내가 처음 만든 컴포넌트'
        }
    }
}
</script>

<style>
    h1 {
        text-decoration: underline;
        color: black;
    }
</style>
```

- date() : 변수를 선언해주는 곳
  - 제이슨 형식으로 여러개의 변수 선언 가능
  - return 문 반드시 필요
