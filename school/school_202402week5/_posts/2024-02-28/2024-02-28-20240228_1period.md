## 2 way binding

```html
<template lang="">
    <input type="email" v-model="email">

    <p>
        <div class="container">
            <div>Email: {{ email }}</div>
        </div>
    </p>
</template>
<script>
export default {
    data: function() {
            return {
                email:'',
            }
    }  
}
</script>
<style lang="">
    
</style>
```

- email:'' : 자바 스크립트에서 email 변수 생성
- v-model="email" : 사용자가 입력한 값을 변수에 저장
- Email: {{ email }} : 변수에 저장된 값을 출력
