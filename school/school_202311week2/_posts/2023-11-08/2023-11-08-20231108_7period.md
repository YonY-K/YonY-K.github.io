- 웹프로그램에서 주로 사용되는 처리방식(CRUD)
  - Create: 정보 생성
  - Read  : 정보 읽어오기
  - Update: 정보 변경
  - Delete: 정보 삭제

- 배열의 CRUD
    ```java
    int[] score = new int[5];
    score[0] = 5;            //Create
    int myscore = score[0];  //Read
    score[0] = 10;           //update
    score[0] = 0;            //Delete
    //원래 Delete는 공간 자체를 없애야함
    ```
